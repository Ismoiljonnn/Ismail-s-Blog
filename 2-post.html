    YouTube Clone loyihasi haqida

    Men HTML va CSS bo‘yicha o‘rganganlarimni mustahkamlash uchun YouTube Clone nomli loyihani yaratdim. Bu loyiha YouTube’ning asosiy interfeysini — header, qidiruv satri, yon panel va video kartalar gridini — statik tarzda HTML va CSS bilan qayta yaratish tajribasidir.

    Bu loyiha orqali men web sahifa tuzilmasi, CSS bilan layout qilish, flexbox yoki grid singari zamonaviy tartib qo‘llanmalari va elementlar orasidagi joylashuv qoidalarini amalda qo‘llashni o‘rgandim. Sahifa hali to‘liq interaktiv emas, lekin uning ko‘rinishi real YouTube interfeysi kabi tuzilgan — bu UX va UI haqida chuqurroq tushuncha hosil qilishimga yordam berdi.

    Ushbu cloneni men o‘zim loyihalab, GitHub’ga yukladim va bu orqali HTML/CSS bo‘yicha qobiliyatimni mustahkamladim.

    Kodga va to‘liq loyiha tuzilishiga GitHub orqali kirishingiz mumkin: https://github.com/Ismoiljonnn/Youtube-clone
